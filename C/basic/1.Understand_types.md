# 타입의 이해

## C언어의 기본 타입

<p></p>
C언어의 4가지 기본 타입

- 정수 타입: char와 int, 실수 타입: float와 double
- short, long, unsigned, register, volatile 등은 한정자(no type)

## 정수의 내부구조

<p></p>
char: 1byte의 정수를 저장하는 타입

```C
    int main() 
    {
        char ch = 250;
        printf("%d\n", ch);   //print: 1111 1010  : -6
    }
```
